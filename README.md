Repositori ini berisi source code dan laporan untuk Tugas Besar Mata Kuliah Kecerdasan Buatan. Proyek ini mengimplementasikan Artificial Neural Network (ANN) secara manual (from scratch) menggunakan NumPy dan membandingkannya dengan model dari framework TensorFlow/Keras serta Scikit-Learn untuk klasifikasi Dry Bean Dataset.

Struktur Repositori
* `src/` : Berisi file Jupyter Notebook (`ANN_from_scratch.ipynb`) dan dataset yang digunakan.
* `doc/` : Berisi dokumen Laporan Akhir (PDF) yang menjelaskan detail arsitektur, perhitungan, dan analisis eksperimen.
